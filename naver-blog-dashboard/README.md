# 🛠️ 충주미세방충망 블로그 자동화 대시보드 (Windows 데스크톱 앱)

본 프로그램은 **충주미세방충망**(`010-6261-0930`)의 시공 현장 사진을 업로드하여 연락처 워터마크를 자동으로 입히고, AI를 통해 네이버 최적화 포스팅 글을 자동 생성하여 네이버 스마트에디터로 원클릭 기입해주는 Windows 단독 실행형 데스크톱 프로그램(Electron 기반)입니다.

동민 디자인 스킬의 **Warm Editorial (따뜻하고 세련된 크림 테마)** 콘셉트를 사용하여 제작되었습니다.

---

## 1. 개발자/관리자 구동 및 패키징 방법

본 프로그램을 테스트하고 배포용 `.exe` 설치 파일로 빌드하기 위해 아래의 단계를 수행합니다.

### 1.1. 의존성 패키지 설치
프로젝트 폴더 내에서 터미널을 열고 npm 의존성 라이브러리를 설치합니다.
```bash
npm install
```

### 1.2. 로컬 테스트 실행
개발 모드로 앱을 실행하여 작동을 검증합니다.
```bash
npm start
```

### 1.3. Windows 실행 파일 (.exe) 배포 빌드
친구분에게 전달할 단일 설치 파일(`.exe`)을 생성합니다.
```bash
npm run dist
```
* 빌드가 완료되면 `dist/` 폴더 내부에 `naver-blog-dashboard Setup 1.0.0.exe` 파일이 생성됩니다. 이 파일을 친구분에게 전달해 주시면 됩니다.

## 2. 친구분(사용자) 사용 가이드

### 2.1. 프로그램 설치 및 최초 실행
1. 전달받은 `naver-blog-dashboard Setup 1.0.0.exe` 파일을 실행하여 설치를 완료합니다.
2. 설치가 끝나면 바탕화면에 생성된 **'충주미세방충망 블로그 자동화 대시보드'** 바로가기를 통해 프로그램을 켭니다.

### 2.2. 환경 설정 (최초 1회만 진행)
1. 프로그램 우측 상단의 **톱니바퀴(설정) 아이콘**을 클릭하여 환경설정 패널을 엽니다.
2. 발급받으신 **Gemini AI API Key**를 입력합니다.
3. 대표번호(`010-6261-0930`)와 기본 업체명(`충주미세방충망`)이 올바르게 들어가 있는지 확인하고 **[설정 저장하기]**를 클릭합니다. (저장된 정보는 컴퓨터에 영구 기록되어 다음 실행 시 자동으로 불러와집니다)

> **💡 Gemini API Key 발급받는 방법**:
> 1. [구글 AI 스튜디오](https://aistudio.google.com/)에 구글 계정으로 로그인합니다.
> 2. **[Get API key]** 버튼을 누른 뒤, **[Create API key]**를 클릭하여 키를 생성하고 복사합니다.

## 3. 포스팅 작성 순서

1. **시공 정보 설정**:
   * 타겟 노출 모드(기본형 C-Rank 추천), 시공한 도시(충주, 청주, 제천 등), 시공한 아파트 단지명을 입력합니다.
   * 필요시 하단의 서브 키워드(예: `고양이방충망`, `미세방충망비용`)를 추가로 적습니다.
2. **현장 사진 등록 및 편집**:
   * 당일 작업하면서 찍어온 사진들을 마우스로 끌어다가 드롭존에 업로드합니다 (최대 10장).
   * 사진이 업로드되면 자동으로 우측 하단에 `충주미세방충망 010-6261-0930` 워터마크가 투명하게 오버레이됩니다.
   * 각 사진 카드의 `[전]` / `[후]` 단추를 누르면 이미지 상단에 `[시공 전]` / `[시공 후]` 표시가 예쁘게 각인됩니다.
3. **포스팅 생성**:
   * **[블로그 포스팅 글 작성]** 버튼을 누르면 AI가 약 20~30초간 글을 작성합니다.
   * 작성이 완료되면 하단에 제목과 본문(Rich HTML)이 나타나며, 실시간 SEO 채점 지표를 통해 글자 수 및 키워드 반복 횟수가 적절한지 자동으로 채점해 줍니다.
4. **네이버 스마트에디터로 내보내기**:
   * 우측 화면의 네이버 블로그 스마트에디터에서 네이버 계정으로 로그인을 진행하고 **[글쓰기]** 창을 띄웁니다.
   * 좌측 대시보드 하단의 **[블로그로 기사 내보내기]** 단추를 누르면 제목과 본문 서식(HTML)이 오른쪽 네이버 작성창에 자동으로 채워집니다.
   * 본문에 생성된 빨간색 이미지 가이드 위치(예: `[여기에 시공 전 부식된 알루미늄 방충망 사진을 드래그해서 넣어주세요]`)를 확인합니다.
   * 좌측 사진 목록에 있는 썸네일 카드를 마우스로 꾹 누른 채 우측 네이버 화면의 가이드 위치로 **끌어다 던져(Drag & Drop)** 사진을 넣습니다.
   * 사진 매핑이 다 끝나면 본문 내의 빨간색 가이드 안내 문구는 깔끔하게 지워줍니다.
   * 우측 상단의 **[임시저장]** 버튼을 눌러 발행 대기 상태로 글을 보관하고 검토 후 발행합니다.
