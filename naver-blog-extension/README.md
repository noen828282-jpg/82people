# 🛠️ 충주미세방충망 블로그 자동화 도우미 (크롬 확장 프로그램)

본 프로그램은 **충주미세방충망**(`010-6261-0930`)의 청주, 충주, 제천 등 충청 지역 시공 후기를 네이버 블로그에 최적화하여 손쉽게 작성할 수 있도록 돕는 크롬 확장 프로그램입니다.

---

## 1. 초기 설치 방법 (1회만 진행)

본 확장 프로그램은 구글 크롬(Chrome) 브라우저 또는 네이버 웨일(Whale) 브라우저에서 사용할 수 있습니다.

### 1단계: 확장 프로그램 관리 페이지 접속
1. 크롬 브라우저를 켜고 주소창에 `chrome://extensions` 입력 후 엔터를 칩니다. (웨일 브라우저의 경우 `whale://extensions`)
2. 우측 상단에 있는 **[개발자 모드]** 토글 스위치를 **켬(활성화)**으로 변경합니다.

### 2단계: 폴더 업로드
1. 좌측 상단에 나타나는 **[압축해제된 확장 프로그램을 로드합니다.]** 버튼을 클릭합니다.
2. 다운로드받으신 `naver-blog-extension` 폴더를 통째로 선택하여 업로드합니다.
3. 목록에 **'충주미세방충망 블로그 자동화 도우미'**가 보이면 설치가 완료된 것입니다.

---

## 2. 사용 방법

### 1단계: API 키 및 업체 정보 설정 (최초 실행 시)
1. 크롬 브라우저 우측 상단 퍼즐 조각 아이콘(확장 프로그램)을 클릭하고 **'충주미세방충망 블로그 자동화 도우미'**의 고정(핀) 아이콘을 눌러 항상 보이게 설정합니다.
2. 툴바에 추가된 파란색 방패 모양 아이콘을 클릭하면 우측에 사이드패널이 열립니다.
3. 사이드패널 상단 **톱니바퀴(환경 설정) 아이콘**을 누릅니다.
4. 발급받으신 **Gemini API Key**를 입력하고 **[설정 저장하기]**를 누릅니다. (한 번 저장하면 계속 유지됩니다)

> **💡 Gemini API Key 발급 방법**:
> 1. [구글 AI 스튜디오](https://aistudio.google.com/)에 접속하여 구글 계정으로 로그인합니다.
> 2. **[Get API key]** 버튼을 클릭하여 새로운 API 키를 생성합니다.
> 3. 생성된 키를 복사하여 프로그램 환경 설정에 입력해 주세요. (무료 사용량 제공)

---

## 3. 포스팅 작성 및 전송하기

### 1단계: 시공 정보 입력
* **SEO 알고리즘 모드**: 포스팅 성격에 맞게 모드를 고릅니다 (C-Rank 시공일지 권장).
* **시공 도시 및 아파트명**: 예) `충주`, `푸르지오 1차` 와 같이 입력합니다.
* **추가 키워드**: `미세방충망가격`, `고양이방충망` 등 추가로 노출하고 싶은 단어를 쉼표로 구분해 적습니다.

### 2단계: 당일 현장 사진 등록 및 워터마크 편집
1. 오늘 현장에서 스마트폰으로 직접 촬영한 시공 전/후 사진을 드래그하여 **[시공 사진 업로드]** 영역에 넣습니다. (최대 10장)
2. 업로드하면 자동으로 이미지 우측 하단에 `충주미세방충망 010-6261-0930` 워터마크가 투명하게 입혀집니다.
3. 각 사진 카드에서 **[시공전]**, **[시공후]** 버튼을 눌러 라벨을 선택하면 이미지 상단 중앙에 예쁘게 라벨이 추가됩니다.

### 3단계: 글 생성 및 네이버 에디터 전송
1. **[블로그 최적화 포스팅 생성]** 버튼을 누릅니다. 약 20~30초 후 본문이 자동으로 완성됩니다.
2. 왼쪽 브라우저 화면에서 **네이버 블로그 글쓰기 페이지(스마트에디터 One)**를 열어둡니다. (로그인 필수)
3. 사이드패널 맨 하단 **[N 네이버 스마트에디터로 전송]** 버튼을 누르면 제목과 본문이 네이버 편집창에 서식을 유지한 채로 한 번에 입력됩니다.
4. 본문 내용 중 빨간색 가이드 문구(예: `[여기에 시공 전 부식된 알루미늄 방충망 사진을 드래그해서 넣어주세요]`)가 표시된 위치에, 사이드패널에 있는 사진 카드를 마우스로 **드래그하여 에디터 안으로** 끌어다 던집니다.
5. 이미지 매핑이 끝나면 빨간색 가이드 문구를 지우고, 네이버 에디터 우측 상단의 **[임시저장]**을 누른 후 발행을 진행합니다.
