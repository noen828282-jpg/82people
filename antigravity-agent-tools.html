공통 스킬: Global Venture Builder (돈이 되는 웹사이트 구축 비법)

모든 에이전트의 시스템 베이스로 작동하는 수익성 강화 스킬셋

단순 자동화를 넘어 전 세계 영어권 사용자들을 대상으로 직접적인 매출을 발생시키기 위해, 동민 AI 컴퍼니의 모든 에이전트가 공통적으로 신뢰하고 활용하는 기술 아키텍처 및 기획 전략입니다.

  • 글로벌 퍼스트 기획 (Global-First): 모든 웹 UI 텍스트와 상품 설명의 디폴트를 영어로 설정하여 즉시 글로벌 마켓(Etsy, Amazon 등) 타겟 가능.
  • 서버리스 고수익 프론트엔드: 고비용의 복잡한 백엔드 없이 HTML/JS 및 Tailwind CSS, React 환경에서 고성능 정적 웹 빌드 및 Netlify/Vercel 호스팅 배포.
  • Firebase 연동 실시간 저장 (Firebase MCP): Firestore 데이터베이스와 Auth 회원가입 연동을 완벽하고 단순하게 코드에 내장하여 사용자 정보 영구 관리.
  • PayPal API 결제 체크아웃 (PayPal MCP): 스마트 결제 버튼 및 디지털 굿즈 전용 결제 플로우를 적용해, 샌드박스 검증부터 정식 Live 배포까지 결제 즉시 다운로드되는 자동화 구축.
