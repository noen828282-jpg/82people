# Node 16: Next Experiment Recommendation (다음 실험 추천)

이 노드는 쇼핑 쇼츠 자동화 파이프라인의 최종 노드입니다. 이전 단계에서 축적된 미디어 성과 로그 히스토리를 분석하여, 가장 높은 반응 및 커머스 수익을 유도한 카테고리를 추려내고, 다음 파이프라인 주기(Node 1)에서 활용할 최적의 상품 탐색 키워드를 피드백 추천하는 역할을 수행합니다.

## 1. 주요 기능 및 요구사항
1. **성과 데이터 수집 및 분석**:
   * 로컬 히스토리 데이터베이스 파일(`nodes/assets/output/performance_history.json`)을 조회합니다.
   * 각 카테고리별로 총 조회수, 총 링크 클릭률, 총 수익금(Revenue)을 연산하고 합산합니다.
2. **최고 성과 카테고리 판정 (`best_performing_category`)**:
   * 누적 수익금(`revenue_krw`)이 가장 높은 카테고리를 1위 성과군으로 지정합니다.
   * 만약 수익금이 동일하거나 없을 경우, 총 조회수(`views`)가 가장 높은 카테고리를 기준으로 잡습니다.
   * 히스토리가 아예 없거나 데이터가 비어있을 때는 기본값(예: `keyboard`)을 최우선 성과 카테고리로 매핑합니다.
3. **스마트 파생 키워드 추천 (`recommended_keywords`)**:
   * 최고 성과 카테고리에 상응하는 연관 롱테일 키워드 3~5개를 매핑 데이터 테이블을 통해 자동 도출합니다.
   * 이 추천된 키워드 목록은 파이프라인의 다음 루프 시 Node 1 (`collect_products.py`)의 검색어 후보로 즉시 전달됩니다.
4. **추천 근거 보고서 작성**:
   * 어떠한 실적 데이터를 바탕으로 이 키워드들을 선정하게 되었는지 판정 원인(`experiment_rationale`)을 메타데이터로 남깁니다.

## 2. 카테고리별 연관 추천 키워드 매핑 테이블
* **Keyboard (키보드)**:
  - `저소음 적축 키보드`, `인체공학 키보드`, `미니 텐키리스 키보드`, `사무용 무선 키보드`
* **Mouse (마우스)**:
  - `버티컬 무선 마우스`, `무소음 마우스`, `게이밍 경량 마우스`, `휴대용 슬림 마우스`
* **Desk Mat (데스크 매트)**:
  - `가죽 데스크 매트`, `펠트 장패드`, `양면 데스크 장패드`, `방수 마우스 장패드`
* **기타 카테고리 (Default)**:
  - `데스크테리어 꿀템`, `책상 정리 수납함`, `비전기 모니터 받침대`

## 3. 데이터 인터페이스

### 입력 데이터
* Node 15의 성과 지표 로깅이 갱신 완료된 단일 상품 객체를 수신합니다.

### 출력 데이터
* 피드백 루프 결과 객체 `next_experiment`가 추가됩니다.
  - `analyzed_at`: 분석 완료 일시 (ISO 8601 타임스탬프)
  - `best_performing_category`: 판정된 우수 카테고리 (STRING)
  - `recommended_keywords`: 차기 수집용 추천 검색 키워드 목록 (ARRAY of STRING)
  - `experiment_rationale`: 추천 상세 이유 기술서 (STRING)
