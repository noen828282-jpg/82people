# Node 5: Affiliate Link Audit (제휴 링크 감사 노드)

## 1. 개요
* **목적**: 리스크 검증을 무사히 통과한 상품을 대상으로 오리지널 URL을 수익화 제휴 링크(Coupang Partners 추천인 링크 등)로 변환하고, 현행 법규(공정거래위원회 표시광고 심사지침)에 부합하는 명확한 경제적 대가 고지 문구를 생성합니다.
* **트리거 방식**: 규제 검사 완료 트리거 수신 시 작동.
* **주요 흐름**: 리스크가 없는(`risk: false`) 상품 필터링 -> 원본 URL을 제휴 쇼핑 링크로 매핑 -> 공정위 지침에 부합하는 고지 문구(`disclosure`) 생성 및 매핑 -> 결과 저장 및 다음 노드 전달.

## 2. 공정거래위원회 표시광고 심사지침 준수
공정위 개정 지침에 따라 아래 규칙을 적용해 고지 문구를 생성합니다.

* **잘못된 예 (조건부/애매한 표현 - 금지)**:
  * `이 포스팅은 소정의 수수료를 제공받을 수 있습니다.` (수수료 수취 여부가 불명확하므로 사용 불가)
* **올바른 예 (명확한 확정 표현 - 의무)**:
  * `본 영상은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.`
  * `#광고 #쿠팡파트너스` (태그나 자막으로도 명확히 노출되어야 함)

## 3. 인터페이스 정의
* **입력 데이터**: `input_schema.json` 참조 (규제 검사를 마친 상품 중 `risk: false`인 상품 배열)
* **출력 데이터**: `output_schema.json` 참조 (각 상품 객체에 제휴 링크 `affiliate_url` 및 법적 고지 문구 `disclosure`가 결합된 배열)

## 4. 실행 방법
```bash
python audit_affiliate_links.py --input sample_input.json --output output.json
```
