# 쇼츠 업로드 패키지 수동 검수 체크리스트 (Approval Checklist)

아래 항목들을 수동 점검하고 최종 승인 처리를 결정해 주세요.

## 🎬 1. 영상 자산 검수 (Video Quality)
- [ ] 각 장면에 블러(Blur), 깨짐(Glitch), 또는 비정상적인 워터마크가 없는지 확인했나요?
- [ ] 전체 비디오 비율이 9:16 세로형 쇼츠 규격(1080x1920)을 확실히 만족하나요?
- [ ] 씬 1, 씬 2, 씬 3 간 화면 컷 전환이 어색하지 않고 자연스러운가요?

## ✍️ 2. 자막 및 텍스트 검수 (Subtitles & Fonts)
- [ ] 생성된 `.srt` 자막 내용 중 오타, 맞춤법 오류, 어색한 한국어 표현이 없는지 점검했나요?
- [ ] 자막 표출 시점(싱크)이 비디오 음향 또는 전환 흐름과 잘 맞아떨어지나요?
- [ ] 공정위 고지 자막(투명성 확보)이 화면 하단에 너무 작거나 잘리지 않고 정상 배치되나요?

## 🔗 3. 링크 및 설명글 검수 (Description & Links)
- [ ] 본문에 첨부된 제휴 링크(`affiliate_url`)가 실제 동작하며 정확한 타겟 제품으로 이동하나요?
- [ ] 본문 설명글 하단에 공정위 고지 문구가 법적 요건에 맞추어 명확하게 기입되어 있나요?
- #데스크테리어 #키보드추천 등 관련 해시태그에 스팸이나 유해한 키워드가 섞여있지 않나요?

---

### [최종 판정 가이드]
* **모두 통과 시**: 스크립트 실행 후 승인(`Approve`) 입력 $\rightarrow$ Node 15(퍼블리싱/성과측정) 단계로 자동 이행.
* **보류/반려 시**: 반려 사유 기록 후 거절(`Reject`) 입력 $\rightarrow$ 파이프라인 중단 및 영상/대본 재생성 큐로 인입.
